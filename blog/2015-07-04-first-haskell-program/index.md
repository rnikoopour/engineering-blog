For a while I've wanted to start writing in a functional language. However, I never really knew how to start or what to do. I read most of the way through [Learn You A Haskell](http://learnyouahaskell.com/). Then I read about a functional language called [Erlang](http://www.erlang.org/) which looked really cool. I read a bit of [Learn You Some Erlang](http://learnyousomeerlang.com/). Things made a bit more sense, but I still didn't know what to do.

Yesterday, I finally wrote my first program in [Haskell](https://www.haskell.org/). It's called [Python Class Creator](https://github.com/rnikoopour/Python-Class-Creator). Python Class Creator is a pretty simple program. It takes in two strings and then dumps out a basic python class implementation. Check it out on my [Github](https://github.com/rnikoopourtab=repositories) page.
